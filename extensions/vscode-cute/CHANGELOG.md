# Changelog

## 0.1.0

- Initial release.
- TextMate grammar covering keywords (`prop` / `signal` / `slot` /
  `view` / `widget` / `arc` / `class` / `struct` / `fn` / `let` /
  `var` / `pub` / `extern value` / `escaping` / `consuming` /
  `~Copyable` etc.), strings with `#{...}` interpolation, `:symbol`
  literals, comments, numbers, and PascalCase type heuristics.
- `language-configuration.json` for bracket / quote auto-closing
  and `#` line comments.
- 2-space indent default for `[cute]`.
